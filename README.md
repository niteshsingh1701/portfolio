## ğŸ¨ Customization

### Changing Colors
Update the CSS custom properties in `css/style.css`:
```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
  --background-color: #your-color;
}
```

### Adding Projects
Edit the projects section in `index.html` and add your project cards following the existing structure.

### Modifying Animations
GSAP animations can be customized in `js/gsap.js`. Adjust timing, easing, and effects to match your preferences.
